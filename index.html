備份與還原

Google Sites 內嵌頁的修改資料會存在目前瀏覽器。換電腦或清除瀏覽器資料可能會消失，建議修改後按「匯出 JSON」備份。

使用提醒

1. 點右上角「編輯區塊」即可修改內容。
2. 修改後會存在本機瀏覽器。
3. 若要讓所有學生看到同一份更新內容，請匯出 JSON 後請我幫你整合成新版 HTML。
4. Google Sites 若擋掉 JavaScript，請改成 GitHub Pages / Netlify 網址嵌入。
